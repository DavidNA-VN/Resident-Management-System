# Census Management Frontend

Trang đăng nhập cho hệ thống quản lý dân cư Tổ dân phố 7 (Phường La Khê) được dựng bằng React + Vite + TailwindCSS.

## Cách chạy
1. Cài Node.js (>= 18) để có `npm`/`pnpm`.
2. Cài phụ thuộc:
   ```bash
   npm install
   ```
3. Chạy dev server:
   ```bash
   npm run dev
   ```

## Ảnh nền
Thêm ảnh nền độ phân giải cao vào `public/images/la-khe-bg.jpg` để hiển thị phần hero phía sau form đăng nhập.
